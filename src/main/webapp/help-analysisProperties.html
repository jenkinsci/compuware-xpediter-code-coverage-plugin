This field allows you to specify Code Coverage analysis properties.

Analysis properties defined in this field will take precedence over any properties defined in the optionally specified Code Coverage analysis properties file above.
It is possible to specify all Code Coverage properties in this field and leave the "Path to analysis properties" input field blank.

# Workspace-relative path(s) to source directories (Optional)
cc.sources=/src

# Code Coverage repository (Required)
cc.repos=pfhjyg0.xv20.reposit

# Code Coverage System (Required)
cc.system=

# Code Coverage Test (Required)
cc.test=

# Code Coverage DDIO Overrides (Optional)
cc.ddio.overrides=
